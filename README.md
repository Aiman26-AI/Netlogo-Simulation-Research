This repository contains a NetLogo simulation developed as part of my research project in Data Science. The simulation was created using **NetLogo version 7.0.1**.

 # Project Overview :

Title : How Fast Do Static Classifiers Fail Under Covariate Drift? An Agent-Based Simulation Study
Created by : Aiman Shahid Kang, Data Science Student at Virtual University of Pakistan
Created on : 20 December, 2025
NetLogo Version : 7.0.1
Goal : To demonstrate my research


# How to Run the Simulation :

1. Download and install NetLogo 7.0.1 from (https://ccl.northwestern.edu/netlogo/download.shtml).
2. Open `simulation.nlogo` in NetLogo.
3. Use the interface to run the simulation and observe the results.
